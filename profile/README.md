# Booman Systems LLC

Booman Systems LLC builds AI-assisted product, automation, and music-tech systems.

Current focus:

- ModeOS: operating system for modern music and creator businesses
- 130 MODE: flagship music-business platform and proof of work
- Sondex: music catalog and media infrastructure
- SpinbookDJ: DJ booking and venue intelligence
- Internal automation for product operations, catalog workflows, and business systems

Core stack:

- Next.js, React, TypeScript
- Supabase, Stripe, Vercel, Cloudflare
- GitHub, OpenAI/Codex, automation workflows

Primary site: https://modeos.app
